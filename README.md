# SecureStorageLabJava

Application Android en Java pour tester :

- SharedPreferences
- EncryptedSharedPreferences
- Internal Storage
- JSON Storage
- Cache Storage
- External App Storage

## Technologies utilisées

- Java
- Android Studio
- AndroidX Security Crypto
- JSON

## Fonctionnalités

- Sauvegarde et lecture des préférences
- Stockage sécurisé des tokens
- Gestion des fichiers internes
- Sauvegarde JSON des étudiants
- Utilisation du cache Android
- Stockage externe app-specific
- Nettoyage des données

## Structure du projet

- prefs
- files
- cache
- external
- model
- ui
